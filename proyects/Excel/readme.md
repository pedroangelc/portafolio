# Reportes y dashboards
Análisis desarrollado en Excel utilizando fórmulas avanzadas, tablas, tablas dinámicas, gráficas dinámicas, segmentación de datos, vincuilación y filtrado avanzado.

## Objetivo
Mostrar la ceratividad para la visualización de los datos y la funcionalidad de los reportes creados

## Habilidades demostradas
- Fórmulas avanzadas (XLOOKUP, funciones lógicas, etc.)
- Tablas dinámicas
- Dashboards
- Validación y depuración de datos
- Gráficas
- Segmentación
- Tablas

## Contenido del archivo
- Hoja ETL: descarga y tras}nsformació de información de CRM y diversos archivos de varias áreas
- Hoja Modelo: Estructuracón mediante tablas, tablas dinámicas y fómulas
- Hoja Reporte / Dashboard: Visalización de rentabilidad por área geográfica (estado, región), gerencia, ejecutivo, centro de proceso, cliente; utilidad anual, calificación de centro proceso según su rentabilidad.
